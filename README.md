# Netflix Movies and TV Shows Analysis | SQL & Tableau

## Project Overview
This project analyzes Netflix’s Movies and TV Shows dataset to uncover content trends, regional patterns, and audience-oriented insights using **SQL for data analysis** and **Tableau for visualization**.

The primary objective is not only to answer analytical questions using SQL, but also to **translate those insights into interactive dashboards** that help stakeholders quickly understand content trends and make data-driven decisions.

---

## Key Objectives
- Understand Netflix content distribution across Movies and TV Shows
- Analyze trends by release year, country, genre, and duration
- Identify top-performing countries, actors, and genres
- Perform keyword-based content categorization
- Build interactive Tableau dashboards for quick trend analysis

---

## Tools & Technologies
- **Database:** MySQL 8+
- **Query Language:** SQL (CTEs, Window Functions, Recursive Queries)
- **Visualization:** Tableau
- **Dataset:** Netflix Movies & TV Shows ([Kaggle](https://www.kaggle.com/datasets/shivamb/netflix-shows))

---

## SQL Analysis Highlights
- Content distribution (Movies vs TV Shows)
- Most frequent ratings by content type
- Year-wise and country-wise content trends
- Genre-wise content breakdown using recursive CTEs
- Actor and director analysis for Indian content
- Keyword-based classification of content as *Good* or *Bad*
- Duration-based analysis for movies and TV shows

Advanced SQL techniques used:
- Recursive CTEs for splitting multi-valued columns
- Window functions for ranking and trend analysis
- String parsing and date transformations
- Conditional aggregation and CASE logic

---

## Tableau Dashboard Highlights
The Tableau dashboard was designed to provide **quick, high-level insights** for decision-making.

Key visualizations include:
- **Pie Chart:** Movies vs TV Shows distribution
- **Trend Analysis:** Content growth over years
- **Top Countries & Genres:** Content concentration analysis
- **Top Actors & Directors:** Popular contributors
- **Keyword-Based Categorization:** Content nature overview
- **Interactive Filters:** Year, country, type, and genre

The dashboard enables users to explore Netflix content trends dynamically without writing queries.

---

## Business Insights
- Netflix content has grown significantly post-2015, peaking around 2019–2020
- Movies dominate the platform, but TV Shows show consistent growth
- The United States and India are the top content-producing countries
- Drama and International content form the largest genre segments
- Keyword-based analysis highlights the proportion of mature content
